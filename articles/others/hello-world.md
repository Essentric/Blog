### 在这里写出所有我知道的输出 `Hello World` 的方式, 后面如果学到了新的方式,再来补充.

* C 语言:

``` c
printf("Hello World!");
```

* C++

``` c++
cout << "Hello World!";
```

* C#
``` c#
System.Console.Write("Hello World!");
```

* Java

``` java
System.out.println( "Hello World!" );
```

* PHP

``` php
echo "Hello World!";
```

* JavaScript

``` javascript
console.log("Hello World");
```
